# Product Overview

This is a **Devvit React Application** - a real-time chat application built with React that runs on Reddit

## Core Features

- Real-time messaging with instant message broadcasting
- Post-based chat rooms - each Reddit post becomes its own chat room
- Automatic Reddit authentication - no login required
- Persistent message history stored in Redis
- Responsive design with desktop sidebar and mobile bottom navigation
- Multiple slide-out panels for navigation, messages, and notifications

## Platform Integration

- Runs as a Reddit app within the Devvit ecosystem
- Creates posts automatically on app installation
- Provides moderator menu actions for post creation
- Uses Devvit's realtime API for live message broadcasting
- Integrates with Reddit's authentication system

## Target Use Case

This chat application will be published and used on Reddit for community discussions, AMAs, live events, and real-time conversations within specific posts
